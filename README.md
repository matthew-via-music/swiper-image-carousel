## Image carousel using the 'Swiper' depdendency

## Features

- Infinite loop back and forth
- Keyboard control with arrow keys
- Mobile touch swipe
- Clickable arrows on screen
- Clickable Dot pagination for images length
- Responsive
- Full screen images
